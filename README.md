# 🌉 DreamSetu

> **Learn Together • Build Together • Grow Together**

DreamSetu is an AI-powered collaborative learning platform that connects students of different skill levels into balanced learning groups called **Setu Pods**. Instead of learning alone through endless tutorials, students build real-world projects together while receiving personalized AI guidance.

---

# 🎯 Vision

To make quality mentorship and collaborative learning accessible to every student, regardless of their background, by creating an AI-powered ecosystem where learners grow together through real projects.

---

# ❗ Problem Statement

Millions of students struggle to learn because they:

- 😟 Fear asking questions
- 🤯 Get overwhelmed by too many learning resources
- 😔 Don't have mentors
- 🤝 Can't find project teammates
- 📉 Lose confidence through comparison
- ❌ Learn theory but never build projects

Traditional platforms teach individually.

DreamSetu believes students learn best **by building together.**

---

# 💡 Solution

DreamSetu combines

- 🤖 AI Mentorship
- 👥 Collaborative Learning
- 💻 Project-Based Learning
- 📊 Skill Assessment
- 🚀 Career Roadmaps

into one intelligent platform.

The platform automatically creates balanced learning teams called **Setu Pods**, where beginners, intermediate learners, and experts collaborate on real-world projects.

---

# ✨ MVP Features

## 🤖 AI Skill Assessment

Analyze student skills using AI.

Outputs:

- 🌱 Beginner
- 🚀 Intermediate
- 🎯 Expert

---

## 👥 AI Setu Pods

Automatically forms balanced teams.

Each pod contains:

- 1 Beginner
- 1 Intermediate
- 1 Expert

Matching factors:

- Skills
- Interests
- Goals
- Language
- Availability

---

## 💻 Project-Based Learning

Students learn while building projects like:

- Portfolio Website
- Weather App
- Expense Tracker
- Chat Application
- To-Do App

---

## 🧩 AI Task Distribution

AI assigns personalized tasks based on skill level.

Example:

### Beginner

- HTML
- CSS
- Images

### Intermediate

- JavaScript
- Responsive UI
- Animations

### Expert

- Performance
- Accessibility
- Code Review
- Mentorship

---

## 🤖 AI Mentor

Instead of giving direct answers,

AI:

- Gives hints
- Explains concepts
- Suggests resources
- Encourages problem solving
- Tracks improvement

---

## 📊 Growth Dashboard

Track:

- Skills Learned
- Projects Completed
- Contribution Score
- Collaboration
- Leadership
- Confidence

---

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/DreamSetu.git
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

---

# 🌍 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
```

# DreamSetu UI Wireframes

## Goal

Design a simple, modern, and user-friendly interface that helps students collaborate, learn, and build projects without fear.

The UI should be clean, minimal, and easy to use, especially for beginners.

---

# Design Principles

- Minimal and distraction-free
- Beginner-friendly
- Mobile responsive
- Accessibility focused
- Collaboration first
- AI assistance always available
- Progress visible everywhere

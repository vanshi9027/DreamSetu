# DreamSetu MVP Features

## MVP Goal

Build a working prototype that demonstrates how AI-powered collaborative learning can help students learn, build, and grow together through real-world projects.

The MVP focuses on solving one core problem:

> "How can beginners, intermediate learners, and experts collaborate effectively while learning?"

---

# Core Feature

## AI-Powered Setu Pods ⭐

DreamSetu's primary innovation is **Setu Pods**.

A Setu Pod is an AI-generated collaborative team consisting of:

- 🌱 Beginner
- 🚀 Intermediate
- 🎯 Expert

AI forms balanced teams based on:

- Skill Assessment
- Learning Goals
- Interests
- Preferred Domain
- Language
- Availability

Every member has a meaningful role in the project.

---

# Feature 1: AI Skill Assessment

## Purpose

Identify each student's current skill level.

### AI Evaluates

- Technical Skills
- Problem Solving
- Communication
- Project Experience

### Output

AI classifies users as:

- 🌱 Beginner
- 🚀 Intermediate
- 🎯 Expert

This helps create balanced Setu Pods.

---

# Feature 2: AI Setu Pod Formation

## Purpose

Automatically create balanced learning teams.

Each Pod contains:

- 1 Beginner
- 1 Intermediate
- 1 Expert

AI matches students using:

- Skills
- Goals
- Interests
- Time Availability
- Preferred Language

---

# Feature 3: Project-Based Learning

Instead of watching tutorials,

students work on real-world projects.

Example Projects:

- Portfolio Website
- To-Do App
- Weather App
- Expense Tracker
- Chat Application

Students learn while building.

---

# Feature 4: AI Task Distribution ⭐

AI divides one project into role-based tasks.

Example:

Project: Portfolio Website

🌱 Beginner

- HTML Structure
- Images
- Buttons
- Basic CSS

🚀 Intermediate

- Responsive Design
- JavaScript
- Animations

🎯 Expert

- Performance
- Accessibility
- Code Review
- Mentorship

Every student contributes according to their current skill level.

---

# Feature 5: AI Mentor

AI acts as a learning companion.

Instead of directly giving answers, AI:

- Gives hints
- Suggests resources
- Explains concepts
- Recommends next steps
- Encourages independent thinking

---

Students learn together while completing projects.

# Why DreamSetu is Different

Most learning platforms focus on **individual learning**.

DreamSetu focuses on **collaborative skill development**.

Instead of learning alone,

students learn by:

- Building together
- Teaching each other
- Solving real-world problems
- Growing through collaboration

AI makes collaboration smarter by creating balanced Setu Pods, assigning personalized tasks, tracking growth, and guiding every learner throughout the journey.

# DreamSetu User Flow

## Goal

Design a seamless learning journey where students move from learning individually to collaborating in AI-powered Setu Pods, building real-world projects, and growing together.

# Step 1 – Landing Page

The student visits DreamSetu.

The landing page introduces:

- DreamSetu Vision
- How Setu Pods Work
- Features
- Success Stories
- Get Started Button

Goal:

Help users understand the platform within 30 seconds.

---

# Step 2 – Registration

Students create an account using:

- Google
- Email
- College ID (Future)

After registration, students complete their profile.

Information:

- Name
- College
- Skills
- Interests
- Preferred Domain
- Preferred Language
- Weekly Availability

---

# Step 3 – AI Skill Assessment ⭐

Every new student completes an assessment.

Assessment includes:

- MCQs
- Coding Questions (if applicable)
- Aptitude
- Communication Questions
- Project Experience

Purpose:

Understand the student's current skill level.

Output:

🌱 Beginner

🚀 Intermediate

🎯 Expert

---

# Step 4 – AI Profile Analysis

AI analyzes:

- Skill Level
- Goals
- Interests
- Domain
- Availability
- Language

AI prepares the student's learning profile.

---

# Step 5 – AI Creates Setu Pod ⭐

DreamSetu automatically forms balanced collaborative teams.

One Setu Pod contains:

🌱 Beginner

🚀 Intermediate

🎯 Expert

AI ensures team members have:

- Similar goals
- Compatible schedules
- Common interests

Each member receives a specific role.

---

# Step 6 – Team Workspace

Each Setu Pod gets a shared workspace.

Workspace includes:

- Team Chat
- Task Board
- Shared Resources
- AI Mentor
- Progress Tracker
- Team Members

Everything needed for collaboration is available in one place.

---

# Step 7 – Project Assignment

AI assigns one real-world project.

Example:

Portfolio Website

Expense Tracker

Weather App

Chat Application

E-Commerce Website

Projects increase in difficulty over time.

---

# Step 8 – AI Task Distribution ⭐

Instead of everyone doing the same work,

AI divides the project into role-based tasks.

Example:

Project: Portfolio Website

🌱 Beginner

- HTML Structure
- Images
- Buttons
- Basic CSS

🚀 Intermediate

- Responsive Layout
- JavaScript
- Animations

🎯 Expert

- Architecture
- Optimization
- Code Review
- Mentorship

Every student contributes according to their level.

---

# Step 9 – Learning & Collaboration

Students work together.

DreamSetu provides:

- Team Chat
- AI Mentor
- Resource Suggestions
- Code Discussions
- Doubt Solving

Learning happens naturally while building.

---

# Step 10 – Task Submission

Each student submits completed tasks.

Intermediate students can review beginner work.

Experts perform the final review.

AI checks:

- Progress
- Participation
- Collaboration

---

# Step 11 – AI Growth Report

After project completion,

AI generates an individual report.

Technical Growth

- Coding
- Problem Solving
- Project Skills

Soft Skills

- Communication
- Leadership
- Collaboration
- Confidence

Students also receive personalized suggestions for improvement.

---

# Step 12 – Contribution Score

Students earn points for meaningful contributions.

Examples:

Complete Task → +20

Ask Good Question → +10

Help Teammate → +15

Explain Concept → +15

Code Review → +20

Mentor Beginner → +25

Complete Project → +50

Contribution matters more than competition.

---

# Step 13 – Level Up

Students unlock new levels based on consistent participation.

Growth Journey

🌱 Explorer

↓

📖 Learner

↓

🛠 Builder

↓

🤝 Collaborator

↓

🎯 Mentor

↓

🏆 Community Leader

---

# Step 14 – Continue Learning

Students join new Setu Pods.

Build more projects.

Become mentors.

Grow continuously.

Learning never stops.

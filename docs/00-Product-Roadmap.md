# DreamSetu Product Roadmap

## Project Overview

DreamSetu is an AI-powered collaborative learning ecosystem that helps students learn, build projects, and grow together through intelligent team formation (Setu Pods), personalized AI guidance, and project-based learning.

Unlike traditional learning platforms that focus only on content, DreamSetu focuses on collaboration, confidence building, mentorship, and real-world skill development.

---

# Vision

To build India's most trusted AI-powered collaborative learning ecosystem where every student can learn, collaborate, build, and grow without fear or comparison.

# Founder Reminder

Before building any new feature, ask:

> Does this feature help students learn, collaborate, build, or grow?

If the answer is **No**, it is **not** part of the MVP.
